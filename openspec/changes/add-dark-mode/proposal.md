## Why

用户在夜间或低光环境下使用应用时，白色背景会造成视觉疲劳。添加深色模式可以提升用户体验，减少眼睛疲劳，并符合现代应用的设计趋势。

## What Changes

- 新增深色模式主题切换功能
- 添加主题切换按钮到导航栏
- 保存用户主题偏好到本地存储
- 所有组件支持深色模式样式

## Capabilities

### New Capabilities
- `theme-toggle`: 主题切换功能，包括切换按钮和状态管理
- `dark-mode-styles`: 深色模式样式系统，包括颜色变量和组件样式

### Modified Capabilities
<!-- 无现有能力需要修改 -->

## Impact

- 前端组件：所有 UI 组件需要添加深色模式样式
- 状态管理：需要新增主题状态管理
- 本地存储：需要保存用户主题偏好
- 样式系统：需要重构为支持主题切换的 CSS 变量系统
